# 🐶 Dog Studio Clone (Three.js)

A simple clone of the **Dog Studio** website built using **Three.js**. This project focuses on recreating smooth 3D interactions, animations, and modern web design using WebGL.

---



---

## 📌 Features

- 🌐 Interactive 3D scene using Three.js  
- 🎥 Smooth camera animations  
- 🖱️ Mouse-based interactions  
- ⚡ Real-time rendering with WebGL  
- 🎨 Minimal and modern UI inspired by Dog Studio  
- 📱 Responsive design (if implemented)

---

## 🛠️ Tech Stack

- Three.js – 3D rendering  
- JavaScript (ES6+)  
- HTML5 & CSS3  
- Vite / Webpack (if used)
